# Game art (optional)

The minigames draw all shapes on Canvas, so no sprite files are required. To
swap in richer art later, drop CC0 packs here and reference them from the
engines (`src/games/flappy`, `src/games/reaction`).

Recommended free **CC0** sources (Kenney.nl — no attribution required):

- Tappy Plane — https://kenney.nl/assets/tappy-plane (flappy obstacles, parallax)
- New Platformer Pack — https://kenney.nl/assets/new-platformer-pack (coins, clouds, pipes)
- Particle Pack — https://kenney.nl/assets/particle-pack (stars, sparkles)
- Digital Audio — https://kenney.nl/assets/digital-audio (jump/coin/score SFX)
